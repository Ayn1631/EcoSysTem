# 生态系统模拟器 (EcoSysTem)

这是一个基于Python的生态系统模拟器，实现了生物的遗传、对抗和学习功能。

## 功能特点

- 基因系统：支持显性/隐性基因的遗传和变异
- 对抗系统：基于属性矩阵的生物间对抗机制
- 学习系统：使用神经网络和强化学习实现生物智能
- 生态模拟：包含资源循环和环境变化的生态系统
- 可视化：实时展示生态系统状态和统计数据

## 安装

1. 克隆项目：
```bash
git clone [项目地址]
cd ecosystem
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用方法

运行主程序：
```bash
python src/main.py
```

## 项目结构

```
ecosystem/
├── src/
│   ├── core/
│   │   ├── gene.py          # 基因系统
│   │   ├── combat.py        # 对抗系统
│   │   ├── neural_network.py # 学习系统
│   │   └── ecosystem.py     # 生态系统
│   └── main.py              # 主程序
├── requirements.txt         # 项目依赖
└── README.md               # 项目说明
```

## 核心模块说明

### 基因系统 (gene.py)
- 实现孟德尔遗传定律
- 支持基因突变
- 显性/隐性性状计算

### 对抗系统 (combat.py)
- 基于属性矩阵的对抗机制
- 动态权重计算
- 平衡性验证

### 学习系统 (neural_network.py)
- 轻量级神经网络
- 强化学习算法
- 知识传承机制

### 生态系统 (ecosystem.py)
- 资源循环系统
- 生物互动机制
- 环境状态管理

## 配置参数

主要参数可以在 `src/main.py` 中调整：
- 生态系统大小 (width, height)
- 初始生物数量 (num_agents)
- 模拟代数 (num_generations)

## 可视化

系统提供两种可视化方式：
1. 实时生态系统状态
   - 资源分布热图
   - 生物位置散点图
2. 统计数据图表
   - 生物数量变化
   - 平均能量变化
   - 平均年龄变化
   - 资源水平变化

## 注意事项
